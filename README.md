# Task Management App (Python CLI)

A simple command-line Task Management Application built using Python.  
This application allows users to add, update, delete, and view tasks using a menu-driven interface.

## Features
- Add tasks
- Update tasks
- Delete tasks
- View tasks
- Exit program

## Technologies
- Python
- CLI
- GitHub

## Run the Project

git clone https://github.com/shivamwaghmare123/task-management-app.git

cd task-management-app

python task.py
